本项目目前仅为学习使用；
项目后端使用Spring, 前端是React，AI框架为Spring AI；
项目预计使用Function call控制前端动画变化，使用RAG增强会话功能，通过提示词优化NPC的表达，控制NPC的“意图”；

游戏目的：
与精神科医生（AI）进行逻辑对抗，证明自己没病；

游戏流程：
会话； NPC会在判断获得了足够的“证据”之后结束会话，并出具“症断书”；

预期拓展：
病类拓展，AI会基于特定的病种进行深入的试探；
NPC角色拓展，会有不同人设的医生，且不同的医生擅长的领域不同；

开发者不是专业的精神科医生，所有的“精神病诊断”皆基于刻板印象以及AI生成，仅供娱乐；
